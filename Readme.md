# NodeJS Blog App

## App Routes

```
/                 => GET  => Homepage route
/life             => GET  => Life posts
/life/:id         => GET  => Returning "Life" post that have ID
/tech-stack       => GET  => Tech stack posts
/tech-stack/:id   => GET  => Returning "Tech stack" post that have ID
/login            => GET  => User login page
/register         => GET  => Get register page
```
